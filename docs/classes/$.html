    chainable

    Defined in C:\working_dir\web-javascript-libraries\src\persgroep.js:74

    lets you attach event listeners to both elements and objects.

    Parameters:

    • eventType String

      An event (or multiple events, space separated) to listen to

    • [selector] String optional

      A CC DOM Element selector string to bind the listener to child elements matching the selector

    • handler Function

      The callback function

    • [args] Arguments optional

      Additional arguments to pass to the callback function

    Example:

    $('.selector').on('click', function() {
    // do something
});

$('.selector').on('click.action', function(e) {
    e.preventDefault();
    // do other stuff
});
